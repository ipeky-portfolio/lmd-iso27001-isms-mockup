# Landesamt für Musterdienste (LMD)
## ISO/IEC 27001:2022 – ISMS Implementation

Dieses Repository dokumentiert die vollständige Implementierung eines Informationssicherheits-Managementsystems (ISMS) gemäß ISO/IEC 27001:2022 für eine Landesbehörde.

## Aufgaben der Behörde

Das Landesamt für Musterdienste (LMD) betreibt zentrale digitale Verwaltungsplattformen und Fachverfahren für Bürgerinnen und Bürger des Landes.

## Ziel des ISMS

- Schutz personenbezogener Daten
- Sicherstellung der Verfügbarkeit digitaler Verwaltungsdienste
- Einhaltung gesetzlicher Anforderungen
- Risikominimierung
- Sicherstellung von Integrität, Vertraulichkeit und Verfügbarkeit

Dieses Projekt dient Demonstrationszwecken zur Darstellung fundierter ISO-27001-Kenntnisse.
